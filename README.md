# Teknofest-HydrodriveZHUM

#  : Hidrojen Yakıt Hücreli Araçlarda Soğuk Başlatma İçin Q-Öğrenme Tabanlı Hibrit Termal Yönetim Sistemi

Bu proje, **Teknofest 2026** kapsamında hidrojen yakıt hücreli araçların dondurucu soğuklarda verimli çalıştırılması için geliştirilmiştir.

## Sistem Bileşenleri
*Yapay Zeka:** Q-Learning algoritması ile optimize edilmiş termal yönetim.
* Hibrit Yapı:** PTC Isıtıcı ve Isı Pompası entegrasyonu  
*Güvenlik:** Akıllı Purge (Tahliye) sistemi ile buzlanma önlem

##  Başarı Metrikleri
* **Enerji Tasarrufu:** Isı pompası optimizasyonu ile yaklaşık %40 verimlilik artışı.
* **Menzil İyileşmesi:** Hibrit strateji sayesinde %15'e varan menzil artışı.
